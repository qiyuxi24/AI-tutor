# 提示词工程入门

## 什么是提示词工程？
**提示词工程（Prompt Engineering）**是指通过精心设计和优化输入给大语言模型的文本（即 Prompt），来引导模型生成准确、高质量、符合预期的结果的过程。

简单来说，就是**“如何更聪明地提问”**。

## 核心思维：结构化提示
一个高质量的提示词通常像写一封专业的邮件，包含以下四个关键要素：

1.  **赋予角色 (Role)**
    *   告诉 AI 它是谁。例如：“你是一位资深 Python 程序员...”
2.  **提供背景 (Context)**
    *   交代前因后果。例如：“我正在开发一个电商网站，用户经常抱怨搜索速度慢...”
3.  **明确任务 (Task)**
    *   清晰地描述你要它做什么。例如：“请帮我优化 SQL 查询语句。”
4.  **设定约束 (Constraints)**
    *   规定输出格式和风格。例如：“请用表格形式回答，并解释每一步优化的原因。”

## 简单示例对比

| 类型 | 提示词示例 | 效果评估 |
| :--- | :--- | :--- |
| ❌ **差提示词** | “帮我写个介绍。” | 模糊不清，AI 只能瞎猜你要介绍什么。 |
| ✅ **好提示词** | “你是一名旅游博主（角色）。我要去成都玩三天（背景）。请列出一份包含美食和景点的行程表（任务），注意不要安排太累的活动（约束）。” | 目标清晰，结果可用性强。 |
