# 递归 vs 迭代

## 基本定义
递归和迭代是两种重复执行逻辑的基本编程范式。

## 对比

| 特性 | 递归 | 迭代 |
|------|------|------|
| **实现方式** | 函数调用自身 | 循环结构 |
| **空间复杂度** | O(n) 调用栈 | O(1) 额外空间 |
| **性能** | 函数调用有开销 | 通常更快 |
| **适用场景** | 树形结构、分治算法 | 线性处理 |

## 相互转换
```python
# 递归版阶乘
def factorial_rec(n):
    return 1 if n <= 1 else n * factorial_rec(n - 1)

# 迭代版阶乘
def factorial_iter(n):
    result = 1
    for i in range(2, n + 1):
        result *= i
    return result
```

## 选择建议
- 问题天然具有递归结构（树、图）时用递归
- 性能敏感或递归深度大时用迭代
