# AI智能体(Agent)基础概念

## 定义
AI智能体（Agent）是指能够感知环境、自主决策并采取行动以实现特定目标的计算机系统。当与大语言模型（LLM）深度结合时，它突破了传统硬编码规则的限制，具备了理解自然语言意图、动态规划任务路径以及调用外部工具的能力。

## 要点
1. **核心特征**：自主性（无需人工逐条干预即可推进任务）、反应性（对环境和输入变化实时感知）、社交性（可与其他Agent或人类进行协作）。
2. **与普通自动化脚本的区别**：脚本是预设的线性指令集（If-Then-Else），遇到未预见异常会直接报错中断；Agent具备推理能力，能根据执行过程中的中间反馈动态调整策略。
3. **与聊天机器人的区别**：ChatBot主要聚焦于多轮对话生成、情感陪伴或信息检索；Agent的核心目标是“完成实际任务”，对话仅是其收集信息和汇报进度的交互手段之一。
4. **典型架构模块**：感知层（接收文本/视觉/环境数据）、规划层（如ReAct/Prompt Chain-of-Thought进行分步推理）、工具调用层（API访问/代码执行/数据库查询）、记忆层（短期会话缓存与长期知识存储）。

## 示例
- **代码执行Agent**：接收数学应用题 -> 自动生成Python求解代码 -> 调用本地沙箱运行 -> 捕获标准输出与错误 -> 综合计算结果返回最终答案。
- **网络调研Agent**：解析用户需求 -> 拆解并并行发起多个搜索引擎请求 -> 抓取Top页面正文 -> 过滤广告噪声 -> 提炼核心观点并结构化输出报告。
