# 烹饪入门

## 一、什么是烹饪？

**烹饪**是通过加热和调味等方式处理食材，将其变成可食用食物的过程。它既是生存技能，也是艺术表达。

### 核心要素
- **食材选择**：新鲜、合适的原料
- **火候控制**：温度与时间的把握
- **调味技巧**：盐、糖、酸、甜等平衡
- **刀工处理**：食材切割的形状与大小

## 二、新手必备工具

| 工具 | 用途 |
|------|------|
| 菜刀 | 切菜、剁肉 |
| 砧板 | 放置食材进行切割 |
| 炒锅/平底锅 | 主要烹饪容器 |
| 铲子/勺子 | 翻炒搅拌 |
| 量杯/秤 | 控制食材分量 |

## 三、安全须知
⚠️ **安全第一**：注意用火、用电安全，避免烫伤和割伤
